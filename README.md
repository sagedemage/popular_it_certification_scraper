# popular_it_certification_scraper
A web scraper program to collect data on the most popular IT certifications.

Install dependencies
```
poetry install
```

Run the main program
```
poetry run python main.py
```

Run the popular_it_certs_by_position program
```
poetry run python popular_it_certs_by_position.py
```

Check for dependency issues
```
poetry run deptry .
```
